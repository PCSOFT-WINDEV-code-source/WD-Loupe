  
<span style="font-family:Arial sans-serif;font-size:16px;">WD Loupe</span>

  
<span style="font-family:Arial sans-serif;font-size:14px;">Cet exemple vous permet de zoomer une partie de l'écran grâce à une loupe visuelle.</span>

<span style="font-family:Arial sans-serif;font-size:14px;">Il est possible de zoomer jusqu'à 8 fois.</span>

<span style="font-family:Arial sans-serif;font-size:14px;">Résumé de l'exemple livré avec WINDEV : </span>

<span style="font-family:Arial sans-serif;font-size:14px;">En utilisant la fonction dCopieImage() du WLangage et la propriété ..Opacité il est possible d'effectuer une capture du bureau Windows à l'emplacement d'une fenêtre sans que celle-ci n'apparaisse dans la copie d'écran.</span>

<span style="font-family:Arial sans-serif;font-size:14px;">Cet exemple utilise cette astuce pour zoomer la partie de l'écran se trouvant en dessous de la loupe. </span>
